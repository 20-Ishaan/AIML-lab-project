# AIML-lab-project
-Loan Status Prediction 
ML Multi-Class Classification Pipeline — Detailed Project Report 
Dataset 
Target Variable 
multiclass_loan_prediction_dataset.csv 
Loan_Status (multiple classes, e.g. Approved / Rejected / Pending etc.) 
. Problem Definition 
The goal of this project is to predict the loan approval status of an applicant based on their financial and 
personal profile. The target column Loan_Status contains multiple discrete class labels (e.g. Approved, 
Rejected, Pending, or similar), making this a multi-class classification problem.

-Salary Prediction 
ML Regression Pipeline — Detailed Project Report 
Dataset 
Target Variable 
Salary_Data.csv 
Salary (continuous numeric — USD) 
. Problem Definition 
The goal of this project is to build a machine learning regression model that can accurately predict an 
employee's salary based on their profile attributes such as years of experience, education level, 
gender, age, and job title. The output is a continuous numeric value (salary in USD), making this a 
regression problem.
